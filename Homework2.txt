В проекті має бути реалізовано функціонал конвертації миль в кілометри, кілометрів в милі. 
Розробку ведіть під Git та GitHub (достатньо однієї гілки master), 
використовуючи Git Bash/Terminal та SSH. Зверніть увагу на створення комітів, формування повідомлень комітів.

(1) Згенеруйте та налаштуйте SSH ключі.
Відкриваємо Git Bash і вводимо команду: ssh-keygen -t ed25519 -C "obslinskij@gmail.com"
Коли спитає куди зберегти ключ - нажимаємо Enter (залишаємо за замовчуванням).
Коли спитає пароль - можна його не вводити і натиснути Enter два рази.
Копіюємо собі ключ і зберігаємо у надійному місці.)
Перевіримо наявність SSH ключів - $ ls ~/.ssh; Отримаємо перелік файлів у поточному каталозі.
Додамо SSH ключі до ssh-agent.
Переконаємося, що ssh-agent працює у фоновому режимі
$ eval `ssh-agent`
Отримаємо таке повідомлення Agent pid (номер)
Це означає, що ssh-agent працює під певним ідентифікатор процесу (PID).
ДОДАМО свій закритий SSH ключ у ssh-agent
$ ssh-add ~/.ssh/id_ed25519
вводимо команду cat ~/.ssh/id_ed25519.pub і копіюємо свій публічний ключ,
після чого переходимо на GitHub → SSH Keys → "New SSH Key" де добавляємо свій ключ.
Тепер Git пов'язаний із GitHub через SSH.
